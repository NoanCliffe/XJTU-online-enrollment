# XJTU线上报道

## 适用范围

**2026年春季学期全日制研究生**~

## 使用方法

### 1. 从源码运行

1. 安装 Python
   请确保系统已安装 Python（版本 3.6 及以上）。

2. 安装依赖包
   
   ```bash
   pip install -r requirements.txt
   ```

1. 运行程序
   在命令行进入程序目录，执行：
   ```bash
   python online_enrollment.py
   ```

### 2. 从二进制运行（Windows）

   访问右侧 Release 页面，下载exe文件双击运行。
